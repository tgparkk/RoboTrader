## 3. 핵심 발견 사항

### ✅ 확인된 사실

1. **시간이 중요하다**
   - 09시 거래가 압도적으로 유리
   - 10시 이후 급격한 성과 하락
   - 현재 시간대 필터가 충분하지 않음

2. **큰 캔들은 위험하다**
   - 급격한 상승(큰 양봉) 이후 매수는 고점 매수 위험
   - 작고 안정적인 상승이 더 안전
   - 현재 이 필터가 없음 (즉시 추가 필요)

3. **모멘텀도 중요하다**
   - RSI 50 이하는 약한 추세
   - 약한 모멘텀에서의 매수는 실패 확률 높음

4. **현재 신뢰도 점수는 효과 없음**
   - 승패 모두 75% 동일
   - 신뢰도 계산 로직 재검토 필요

### ❌ 효과 없는 것들

1. **이등분선 거리** - 승패 차이 미미
2. **시가 대비 상승률** - 승패 차이 미미
3. **저점 대비 거리** - 승패 차이 거의 없음
4. **변동성** - 약간 차이 있으나 유의미하지 않음
