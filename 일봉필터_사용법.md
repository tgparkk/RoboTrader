# 일봉 필터 사용법 (간단 가이드)

## 1줄 요약
**[config/advanced_filter_settings.py](config/advanced_filter_settings.py) 파일에서 `ACTIVE_DAILY_PRESET` 값만 변경하면 됩니다.**

---

## 빠른 시작

### Step 1: 설정 파일 열기
```
D:\GIT\RoboTrader\config\advanced_filter_settings.py
```

### Step 2: ACTIVE_DAILY_PRESET 값 변경 (266번째 줄 근처)

#### 일봉 필터 끄기 (기본값)
```python
ACTIVE_DAILY_PRESET = None
```

#### 일봉 필터 켜기 - 최고 수익 전략 ⭐ 추천
```python
ACTIVE_DAILY_PRESET = 'volume_surge'
```

#### 일봉 필터 켜기 - 최고 승률 전략
```python
ACTIVE_DAILY_PRESET = 'consecutive_2days'
```

### Step 3: 저장하고 봇 재시작
```bash
# 봇이 실행 중이면 중지 (Ctrl+C)
# 다시 시작
python bot.py
```

끝!

## 전략 선택 가이드

| 설정값 | 특징 | 승률 | 수익 (1000만원 기준) |
|--------|------|------|---------------------|
| **'volume_surge'** | 거래량 급증 포착 | 52.7% | **200만원** ⭐ 최고 |
| **'consecutive_2days'** | 2일 연속 상승 | **53.3%** 🏆 | 185만원 |
| 'prev_day_up' | 전일 상승 종목 | 52.7% | 184만원 |
| 'consecutive_1day' | 1일 이상 상승 | 52.8% | 182만원 |
| 'balanced' | 복합 조건 | 52.5% | 173만원 |
| **None** | 필터 없음 | 49.6% ❌ | 144만원 ❌ |

### 추천
- **수익 중시**: `'volume_surge'` (거래량 1.5배 이상 종목만 거래)
- **승률 중시**: `'consecutive_2days'` (2일 연속 상승 종목만 거래)
- **거래 빈도 유지**: `'prev_day_up'` (전일 상승 종목만 거래)

## 주의사항

1. **일봉 데이터 필요**
   - 최초 1회만 수집 필요: `python scripts/collect_daily_for_analysis.py`
   - 이후 자동 갱신됨

2. **3분봉 필터와 독립적**
   - `ACTIVE_PRESET` (3분봉)와 `ACTIVE_DAILY_PRESET` (일봉)은 별개
   - 둘 다 사용 가능

3. **효과**
   - 필터 없음: 승률 49.6%, 수익 144만원
   - volume_surge: 승률 52.7%, 수익 200만원 (+39%)
   - 일봉 필터 사용이 훨씬 유리함
