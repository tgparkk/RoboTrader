## 📊 1. 돌파봉 자체 특성 비교

### ⭐⭐⭐ 1순위: 돌파봉 몸통 크기 (breakout_body_pct)

```
승리: 0.33%
패배: 0.46%
차이: -0.13%p (40% 더 큼!)
```

**분석:**
- 패배 종목의 돌파봉이 **40% 더 큰 양봉**
- 급격한 상승(큰 돌파봉) = 고점 매수 위험
- 작고 안정적인 돌파봉(0.3%대)이 성공 확률 높음

**실전 적용:**
```python
# 돌파봉 몸통이 0.5% 이상이면 차단
if breakout_body_pct >= 0.5:
    return AVOID  # 고점 위험
```

---

### ⭐⭐ 2순위: 돌파봉 거래량 증가 (breakout_volume_vs_prev)

```
승리: 1.43배 (직전 5봉 평균 대비)
패배: 1.20배
차이: +0.23배 (19% 더 높음)
```

**분석:**
- 승리 종목은 돌파봉에서 거래량이 더 많이 증가함
- 돌파봉 거래량이 직전 평균의 1.2배 미만 = 약한 돌파
- 충분한 거래량 뒷받침이 있어야 성공

**실전 적용:**
```python
# 돌파봉 거래량이 직전 평균의 1.3배 미만이면 주의
if breakout_volume_vs_prev < 1.3:
    confidence -= 5  # 신뢰도 감점
```

---

### 3순위: 돌파봉 vs 직전봉 크기 비율 (breakout_vs_prev_body)

```
승리: 1.13배
패배: 1.30배
차이: -0.17배
```

**분석:**
- 패배 종목은 직전봉 대비 돌파봉이 훨씬 크게 나타남
- 갑작스러운 큰 캔들 = 불안정한 신호
- 점진적으로 커지는 패턴이 더 안정적

## 📊 2. 돌파 직전 5봉 패턴 (눌림목 구간)

### 거래량 패턴

#### pre_volume_std (거래량 표준편차)
```
승리: 20,781
패배: 34,362
차이: -13,580 (-65% 더 많음!)
```

**핵심 발견:**
- 패배 종목의 눌림목 구간 거래량 변동성이 **65% 더 큼**
- 거래량이 들쑥날쑥한 눌림목 = 불안정한 패턴
- 안정적으로 거래량이 감소하는 패턴이 성공 확률 높음

**실전 적용:**
```python
# 눌림목 구간 거래량 변동성이 과도하면 차단
if pre_volume_std > 30000:
    return AVOID  # 불안정한 패턴
```

---

#### pre_volume_decreasing_ratio (거래량 하락 추세 비율)
```

```

**분석:**
- 두 그룹 간 차이가 거의 없음
- 거래량 하락 추세 자체보다는 **변동성(표준편차)**이 중요

---

#### pre_min_volume_ratio (최소 거래량 비율)
```

차이: +0.008 (거의 동일)
```

**분석:**
- 두 그룹 모두 눌림목에서 거래량이 평균의 1/3 수준까지 감소
- 최소 거래량 수준은 승패에 큰 영향 없음

### 가격 패턴

- 흔들리는 저점 = 약한 지지

**분석:**
