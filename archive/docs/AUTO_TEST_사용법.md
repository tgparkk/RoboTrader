| 시간 | 동작 |
|------|------|
| 10:30 | 첫 번째 테스트 (과거 데이터 30개 vs 최근 데이터 비교) |
| 12:00 | 두 번째 테스트 |
| 14:00 | 세 번째 테스트 |

## 동작 시간

### 허용 시간
- **평일 06:00~16:00**
- 06:00~09:00: 장 시작 전 (대기)
- 09:00~15:30: 장중
- 15:30~16:00: 장 마감 후 대기
- 16:00 이후: 자동 종료

### 차단 시간
- **주말** (토, 일)
- **평일 16:00 이후**
- **다른 날짜** (2026-01-28 이후는 날짜 체크로 종료)
