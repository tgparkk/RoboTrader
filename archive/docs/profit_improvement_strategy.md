## 실행 제안

### 즉시 테스트 가능한 설정

```python
# config/ml_settings.py
ML_THRESHOLD = 0.45  # 50% → 45%

# 새 파일: config/investment_settings.py
class InvestmentSettings:
    """거래당 투자금 설정"""

    BASE_AMOUNT = 1_000_000  # 기본 100만원

    # 시간대별 배율
    HOUR_MULTIPLIER = {
        9: 1.5,   # 09시: 150만원 (승률 88.9%)
        10: 1.0,  # 10시: 100만원 (승률 42.4%)
        11: 1.0,  # 11시: 100만원 (승률 43.8%)
    }

    # ML 확률별 배율
    ML_PROB_MULTIPLIER = {
        (0.7, 1.0): 1.5,   # 70%+: 150만원
        (0.6, 0.7): 1.2,   # 60-70%: 120만원
        (0.5, 0.6): 1.0,   # 50-60%: 100만원
        (0.45, 0.5): 0.8,  # 45-50%: 80만원
    }

    def get_investment_amount(self, hour, ml_prob):
        """투자금 계산"""
        hour_mult = self.HOUR_MULTIPLIER.get(hour, 0.5)

        ml_mult = 1.0
        for (low, high), mult in self.ML_PROB_MULTIPLIER.items():
            if low <= ml_prob < high:
                ml_mult = mult
                break

        # 최종 투자금 (둘 중 큰 배율 적용)
        final_mult = max(hour_mult, ml_mult)
        return int(self.BASE_AMOUNT * final_mult)
```
